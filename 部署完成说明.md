## 🔄 如何更新内容

### 修改WiFi密码或其他信息

1. **编辑HTML文件**
   ```bash
   # 打开 wifi-info.html
   # 修改相应内容
   ```

2. **重新上传到Netlify**
   - 访问：https://app.netlify.com/drop
   - 拖拽新的 `wifi-info.html`
   - 会生成新的URL

3. **重新生成二维码**
   ```bash
   # 修改 generate_qrcode.py 中的URL
   python3 generate_qrcode.py
   ```

4. **提交到GitHub**
   ```bash
   git add .
   git commit -m "更新内容"
   git push
   ```

## 🆘 常见问题

### Q: 二维码扫不出来？
**A**: 
- 确认网络连接正常
- 尝试用其他扫码工具
- 检查二维码图片是否清晰

### Q: 密码无法复制？
**A**: 
- 使用现代浏览器（Chrome、Safari）
- 或使用微信扫码
- 点击密码区域即可复制

### Q: 页面显示不正常？
**A**: 
- 刷新页面
- 清除浏览器缓存
- 或使用无痕模式

### Q: 想修改样式？
**A**: 
- 编辑 `wifi-info.html` 中的 `<style>` 部分
- 重新上传到Netlify
- 重新生成二维码
