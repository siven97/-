# 启用 GitHub Pages - 详细图文步骤

## 🎯 目标
让您的网站通过 `https://siven97.github.io/-/wifi-info.html` 访问

---

## 📝 操作步骤（3分钟）

### 步骤1：打开仓库设置

1. **打开浏览器**，访问您的仓库：
   ```
   https://github.com/siven97/-
   ```

2. **点击顶部的 Settings（设置）标签**
   - 在仓库名称下方的菜单栏中
   - 如果看不到，说明您不是仓库所有者

### 步骤2：找到 Pages 设置

1. **在左侧菜单中向下滚动**
2. **找到并点击 "Pages"**
   - 在 "Code and automation" 部分下
   - 图标是一个小地球

### 步骤3：配置 Pages

1. **在 "Source" 部分**：
   - Branch: 从下拉菜单选择 **main**
   - Folder: 选择 **/ (root)**

2. **点击 Save 按钮**

### 步骤4：等待部署

1. 页面会刷新，显示：
   ```
   Your site is ready to be published at https://siven97.github.io/-/
   ```

2. **等待 1-2 分钟**

3. **刷新页面**，当显示变为：
   ```
   ✅ Your site is live at https://siven97.github.io/-/
   ```
   就成功了！

### 步骤5：测试访问

在浏览器中打开：
```
https://siven97.github.io/-/wifi-info.html
```

如果能看到页面，说明成功！

---

## 🔧 如果遇到问题

### 问题1：找不到 Settings 标签
**原因**：您不是仓库所有者
**解决**：
1. 确认您已登录GitHub
2. 确认这是您的仓库
3. 刷新页面重试

### 问题2：找不到 Pages 选项
**原因**：仓库可能是私有的
**解决**：
1. 在 Settings → General 中
2. 找到 "Danger Zone"
3. 点击 "Change visibility"
4. 改为 "Public"

### 问题3：部署后还是404
**原因**：需要等待时间
**解决**：
1. 等待 5-10 分钟
2. 清除浏览器缓存
3. 使用无痕模式访问
