# 一键部署到在线服务

## 🚀 最简单的方法：使用 GitHub Gist

### 步骤（5分钟完成）：

1. **访问 GitHub Gist**
   ```
   https://gist.github.com
   ```

2. **创建新Gist**
   - 点击右上角的 "+" 号
   - 选择 "New gist"

3. **粘贴代码**
   - Filename: `wifi-info.html`
   - 将 `wifi-info.html` 的全部内容复制粘贴进去
   - 选择 "Create public gist"

4. **获取原始链接**
   - 点击右上角的 "Raw" 按钮
   - 复制浏览器地址栏的URL
   - 类似：`https://gist.githubusercontent.com/用户名/xxx/raw/xxx/wifi-info.html`

5. **生成二维码**
   - 编辑 `generate_qrcode.py`
   - 将 `url` 改为上面复制的URL
   - 运行：`python3 generate_qrcode.py`

6. **完成！**
   - 使用新生成的 `wifi_qrcode.png`

## 🌐 方法二：使用 jsDelivr CDN

### 步骤：

1. **上传到GitHub**
   - 创建仓库并上传 `wifi-info.html`

2. **使用jsDelivr**
   - URL格式：
   ```
   https://cdn.jsdelivr.net/gh/用户名/仓库名@main/wifi-info.html
   ```

3. **生成二维码**
   - 使用上面的URL重新生成二维码

---

## 📦 方法三：使用 Netlify Drop

### 步骤（最快，无需注册）：

1. **访问 Netlify Drop**
   ```
   https://app.netlify.com/drop
   ```

2. **拖拽文件**
   - 直接拖拽 `wifi-info.html` 到页面
   - 等待上传完成

3. **获取URL**
   - 自动生成类似：`https://xxx.netlify.app/wifi-info.html`
   - 复制URL

4. **生成二维码**
   - 使用新URL重新生成二维码

---

## 🎯 推荐顺序

1. **最快**: Netlify Drop（无需注册，拖拽即可）
2. **最稳**: GitHub Gist（永久有效）
3. **最专业**: GitHub Pages（可自定义域名）

### 使用在线二维码生成器

如果您已经部署好HTML，可以直接使用在线工具：

1. **访问**：https://www.qr-code-generator.com/
2. **输入URL**：粘贴您的HTML页面URL
3. **下载二维码**：直接下载使用

---

## 📝 注意事项

1. **Gist的Raw链接**
   - 每次更新Gist，Raw链接会变化
   - 建议使用固定的GitHub Pages

2. **Netlify Drop**
   - 免费版有流量限制
   - 适合小规模使用

3. **测试**
   - 部署后一定要先测试
   - 确认密码复制功能正常

---

## 🆘 遇到问题？

### 问题1：二维码扫不出来
**解决**：确认URL可以在浏览器中直接访问

### 问题2：密码无法复制
**解决**：使用现代浏览器或微信扫码

### 问题3：页面显示不正常
**解决**：检查HTML文件是否完整上传

---

**选择最适合您的方式，开始使用吧！** 🎉
