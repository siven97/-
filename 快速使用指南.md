# 快速使用指南

## ✅ 已生成的文件

1. **wifi-info.html** - 精美的WiFi信息展示页面
2. **wifi_qrcode.png** - 完整版二维码（推荐使用）
3. **wifi_qrcode_simple.png** - 简单版二维码
4. **wifi_direct_qrcode.png** - WiFi直连二维码

## 🚀 三种使用方式

### 方式一：本地测试（最快）

1. 直接用浏览器打开 `wifi-info.html` 文件
2. 测试密码复制功能是否正常
3. 但是**二维码暂时无法使用**（需要部署到服务器）

### 方式二：使用GitHub Pages（推荐，免费）

#### 步骤：

1. **创建GitHub仓库**
   - 访问 https://github.com/new
   - 仓库名称：`wifi-info`（或任意名称）
   - 选择 Public
   - 点击 Create repository

2. **上传文件**
   - 点击 "uploading an existing file"
   - 拖拽 `wifi-info.html` 到页面
   - 点击 "Commit changes"

3. **启用GitHub Pages**
   - 进入仓库的 Settings
   - 左侧菜单找到 "Pages"
   - Source 选择 "main" 分支
   - 点击 Save

4. **获取URL**
   - 等待1-2分钟
   - 页面会显示类似：`https://你的用户名.github.io/wifi-info/wifi-info.html`
   - 复制这个URL

5. **重新生成二维码**
   - 编辑 `generate_qrcode.py`
   - 找到这一行：`url = f"file://{html_file}"`
   - 改为：`url = "https://你的用户名.github.io/wifi-info/wifi-info.html"`
   - 运行：`python3 generate_qrcode.py`
   - 使用新生成的二维码

### 方式三：使用Vercel（最简单，免费）

#### 步骤：

1. **访问Vercel**
   - 打开 https://vercel.com
   - 用GitHub账号登录

2. **创建新项目**
   - 点击 "Add New..." → "Project"
   - 选择 "Import Git Repository"
   - 或直接拖拽文件夹

3. **部署**
   - 上传包含 `wifi-info.html` 的文件夹
   - 点击 "Deploy"
   - 等待部署完成

4. **获取URL**
   - 部署完成后会显示URL
   - 类似：`https://your-project.vercel.app/wifi-info.html`

5. **重新生成二维码**
   - 同方式二的步骤5

## 📱 使用二维码

### 打印使用
1. 打开 `wifi_qrcode.png`
2. 打印出来
3. 贴在需要的地方

### 数字展示
1. 将 `wifi_qrcode.png` 发送到手机
2. 在手机上展示
3. 其他人扫码即可查看

## ⚠️ 重要提示

1. **本地文件路径无法通过二维码访问**
   - 必须部署到Web服务器
   - 推荐使用GitHub Pages或Vercel（都是免费的）

2. **测试二维码**
   - 部署后，先用手机扫码测试
   - 确认页面正常显示
   - 确认密码可以复制

3. **WiFi直连二维码**
   - `wifi_direct_qrcode.png` 是WiFi直连二维码
   - 需要修改脚本中的WiFi名称（ssid）
   - 扫码后可以直接连接WiFi（无需打开网页）
